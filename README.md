# PVP 플러그인

마인크래프트 시참 PVP 콘텐츠를 위한 종합 관리 플러그인입니다. 자원 시간과 킬타임, 자기장 시스템, 팀 대전 등 다양한 기능을 제공하며, 인게임 GUI를 통해 설정을 쉽게 변경할 수 있습니다.

## 주요 기능

### 1. 게임 모드
- **솔로 모드 (Solo)**: 개인전으로 진행되며, 최후의 1인이 승리합니다.
- **팀 모드 (Team)**: 
  - **랜덤 배정**: 게임 시작 시 자동으로 팀이 배정됩니다.
  - **수동 배정**: `/pvp team join` 명령어로 원하는 팀에 들어갈 수 있습니다.
  - 같은 팀끼리는 공격할 수 없습니다 (팀킬 방지).
  - 최후의 한 팀이 남으면 승리합니다.

### 2. 게임 진행
- **시작 아이템**: 게임 시작 시 다음 아이템이 자동 지급됩니다.
  - 경험치 100 레벨
  - 빵 16개
  - 마법 부여대 1개
  - 책장 64개
  - 청금석 64개
- **자원 시간**: 
  - 설정된 시간(기본 30분) 동안 PVP가 금지되고 무적 상태가 됩니다.
  - 광물을 캐면 자동으로 제련된 주괴가 나옵니다.
  - **행운 곡괭이 효과**: 행운 인챈트 레벨에 따라 추가 광물을 획득할 확률이 증가합니다.
  - 나뭇잎을 파괴하면 나뭇잎 대신 확률적으로 사과가 드롭됩니다.
- **킬타임**:
  - 자원 시간이 끝나면 자동으로 시작됩니다.
  - 지하(Y좌표 62 미만)에 있는 플레이어는 지상으로 이동됩니다.
  - 설정된 Y좌표 미만에서는 블록 파괴 금지, 초과에서는 블록 설치가 금지됩니다.
  - 자기장이 주기적으로 줄어들며 플레이어를 압박합니다.

### 3. 편의 기능
- **인게임 설정 (GUI)**: `/pvp config` 명령어로 게임 설정을 클릭만으로 변경할 수 있습니다.
- **탭 리스트**: `/tab config` 명령어로 탭 리스트의 헤더와 푸터를 설정할 수 있습니다. (자동 그라데이션 적용)
- **채굴 모니터링**: 플레이어의 광물 채굴 비율을 감시하여 핵 사용 의심 유저를 관리자에게 알립니다.
- **게임 기록**: 게임 종료 시 승자와 소요 시간 등이 기록되며 `/pvp history`로 확인할 수 있습니다.

## 명령어

### 관리자 명령어 (OP 전용)
- **/pvp start**: 게임을 시작합니다.
- **/pvp stop**: 진행 중인 게임을 강제로 종료합니다.
- **/pvp config**: 게임 설정 GUI를 엽니다. (자원 시간, 자기장, 드롭 확률, 게임 모드 등)
- **/tab config**: 탭 리스트 설정 GUI를 엽니다.
- **/pvp info <플레이어>**: 해당 플레이어의 채굴 정보와 인벤토리를 확인합니다.

### 플레이어 명령어
- **/pvp team join <팀번호>**: (팀 모드/수동 배정 시) 해당 팀에 참가합니다.
- **/pvp team list**: 현재 팀 현황을 확인합니다.
- **/pvp history**: 최근 게임 기록을 확인합니다.
- **/top**: 게임 중 1회에 한해 지상으로 이동할 수 있습니다.
- **/pvp help**: 도움말을 확인합니다.

## 설치 및 설정
1. 플러그인 파일을 서버의 `plugins` 폴더에 넣고 서버를 실행합니다.
2. `config.yml` 파일에서 세부 설정을 변경하거나, 인게임에서 `/pvp config`를 사용하세요.
