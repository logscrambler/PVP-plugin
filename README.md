# PVP 플러그인

마인크래프트 시참 PVP 콘텐츠를 위한 플러그인입니다. 자원 시간과 킬타임, 그리고 자기장 시스템을 통해 긴장감 넘치는 게임을 진행할 수 있습니다.

## 주요 기능

### 1. 게임 진행 시스템
게임은 크게 자원 시간과 킬타임으로 나뉩니다.

- **자원 시간**: 
  - 게임 시작 후 20분 동안 진행됩니다.
  - PVP가 금지되며, 플레이어는 데미지를 입지 않습니다.
  - 야간 투시와 포화 효과가 무제한으로 적용됩니다.
  - 광물을 캐면 자동으로 제련된 주괴가 나옵니다 (철, 금, 구리).
  - 나뭇잎을 파괴하면 확률적으로 사과를 얻을 수 있습니다.

- **킬타임**:
  - 자원 시간이 끝나면 자동으로 시작됩니다.
  - 지하에 있는 플레이어들은 자동으로 지상으로 이동됩니다.
  - Y좌표 30 이하에서는 블록을 설치하거나 파괴할 수 없습니다.
  - 자기장이 주기적으로 줄어들며 플레이어들을 압박합니다.

### 2. 자기장 시스템
- 초기 크기는 1000이며, 킬타임이 시작되면 3분마다 줄어듭니다.
- 자기장 밖으로 나가면 지속적인 데미지를 입어 사망할 수 있습니다.

### 3. 스코어보드
화면 오른쪽에 현재 게임 상태를 실시간으로 보여줍니다.
- 게임 시간
- 남은 자원 시간 또는 자기장 축소 시간
- 현재 킬 수 및 남은 플레이어 수
- 현재 자기장 크기
- 본인의 좌표

## 명령어

### 관리자 명령어
- **/pvp start**: 게임을 시작합니다.
- **/pvp stop**: 진행 중인 게임을 강제로 종료하고 초기화합니다.
- **/pvp help**: 명령어 도움말을 확인합니다.

## 설치 및 실행
1. 플러그인 파일을 서버의 plugins 폴더에 넣습니다.
2. 서버를 실행하면 플러그인이 자동으로 활성화됩니다.
3. 관리자 권한(OP)을 가진 플레이어가 /pvp start 명령어로 게임을 시작하면 됩니다.
